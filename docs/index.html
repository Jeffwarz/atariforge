ATARI FALCON 030 · 1992

Atari's final
masterpiece.

The Falcon was Atari's greatest machine — and their last. A Motorola 68030 CPU paired with a Motorola DSP56001 signal processor, 16-bit stereo sound, a 65,536-colour display, and a true 32-bit data bus. No other home computer in 1992 touched it. Configure the Falcon the way it deserved to be.

Capabilities

Built for enthusiasts
who demand precision.

01
Profile Manager

Save and switch between unlimited machine configurations. Each profile is a complete snapshot of your hardware, storage, display, sound, and peripherals.

02
Hardware Configurator

Expose every Hatari config parameter through a clean, organised interface. TOS selection, RAM allocation, blitter, MIDI routing, and more — no config file editing required.

03
Storage Management

Mount GEMDOS directories, floppy images, and ACSI hard disk images. Configure boot priority and manage your entire software library without leaving the app.

04
Cross-Platform Native

Built with Tauri for native performance on macOS, Windows, and Linux. Uses your system's Hatari installation — no bundled emulator, no version lock-in.

05
Display & Sound

Configure monitor type, resolution, frame rate, and CRT emulation. Route audio, set YM2149 mixing, and enable STE DMA or Falcon 16-bit output exactly as the original hardware did.

06
Open Source Forever
